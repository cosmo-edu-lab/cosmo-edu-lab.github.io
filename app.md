An interactive tool to explore the major themes of modern cosmology — from the origin of the universe to dark matter, from cosmic expansion to the microwave background.

Designed to be used in class or independently, with content suited to high school students and anyone curious about the cosmos.

[Download]({{ site.github_repo }}/releases){: .btn .btn-primary} [Read the manual]({{ site.github_repo }}/wiki){: .btn }

## Features

- **Simulations** — visualise cosmological processes interactively.
- **Real data** — work with data from actual astronomical observations.
- **Classroom-ready** — structured learning paths with clear objectives.
- **Verified content** — developed and reviewed by university researchers.
{: .grid}

## Installation

```bash
git clone {{ site.github_repo }}.git
cd cosmo-edu-lab
pip install -r requirements.txt
```
